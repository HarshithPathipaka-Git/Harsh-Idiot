# HTML Entities

## What are HTML Entities?

**Purpose:** Display reserved characters and special symbols in HTML.

**Syntax**

```html
&entity_name;
```

or

```html
&#entity_number;
```

---

## `&copy;` (©)

**Purpose:** Displays the copyright symbol.

**Example**

```html
&copy;
```

**Output**

```
©
```

---

## `&lt;` (<)

**Purpose:** Displays the less-than symbol.

**Example**

```html
&lt;
```

**Output**

```
<
```

---

## `&gt;` (>)

**Purpose:** Displays the greater-than symbol.

**Example**

```html
&gt;
```

**Output**

```
>
```

---

## `&amp;` (&)

**Purpose:** Displays the ampersand symbol.

**Example**

```html
&amp;
```

**Output**

```
&
```

---

## Emoji

**Purpose:** Displays emojis using Unicode.

**Example**

```html
&#128512;
```

**Output**

```
😀
```

Or simply:

```html
😊 🚀 ❤️
```

---

# ⭐ Most Used

- `&copy;`
    
- `&lt;`
    
- `&gt;`
    
- `&amp;`
    

---

# ✅ Best Practices

- Use entities for reserved HTML characters (`<`, `>`, `&`).
    
- Use Unicode or entities for symbols and emojis.
    
- Prefer directly typing emojis when supported.
