
# Semantic HTML ⭐

## `<header>`

**Purpose:** Defines the header of a page or section.

**Use**

- Logo
    
- Website title
    
- Navigation
    

---

## `<nav>`

**Purpose:** Defines navigation links.

**Use**

- Navbar
    
- Menus
    
- Breadcrumbs
    

⭐ Used for website navigation.

---

## `<main>`

**Purpose:** Contains the main content of the page.

**Use**

- Primary page content
    

⭐ Only one `<main>` per page.

---

## `<section>`

**Purpose:** Groups related content.

**Use**

- About
    
- Services
    
- Contact
    
- Features
    

---

## `<article>`

**Purpose:** Represents independent, self-contained content.

**Use**

- Blog posts
    
- News articles
    
- Product cards
    
- Forum posts
    

---

## `<aside>`

**Purpose:** Contains content related to the main content.

**Use**

- Sidebar
    
- Advertisements
    
- Related posts
    

---

## `<footer>`

**Purpose:** Defines the footer of a page or section.

**Use**

- Copyright
    
- Contact information
    
- Social links
    
- Quick links
    

---

## `<figure>`

**Purpose:** Groups media with its caption.

**Use**

- Images
    
- Charts
    
- Diagrams
    
- Code snippets
    

---

## `<details>`

**Purpose:** Creates expandable/collapsible content.

**Example**

```html
<details>
```

**Use**

- FAQs
    
- Hidden information
    

---

## `<summary>`

**Purpose:** Defines the visible heading for `<details>`.

**Use**

- Expand/Collapse title
    

---

## `<time>`

**Purpose:** Represents a date or time.

**Example**

```html
<time datetime="2026-06-27">
```

**Use**

- Blog publish date
    
- Events
    
- Schedules
    

---

## `<address>`

**Purpose:** Displays contact information.

**Use**

- Email
    
- Phone
    
- Company address
    
- Website
    

---

# ⭐ Most Used

- `<header>`
    
- `<nav>`
    
- `<main>`
    
- `<section>`
    
- `<article>`
    
- `<footer>`
    

---

# ⭐ Occasionally Used

- `<aside>`
    
- `<figure>`
    
- `<details>`
    
- `<summary>`
    
- `<time>`
    
- `<address>`
    

---

# ✅ Why Use Semantic HTML?

- Better SEO
    
- Better Accessibility
    
- Cleaner Code
    
- Easier Maintenance
    
- Improves Code Readability
    
- Helps Search Engines Understand Page Structure
