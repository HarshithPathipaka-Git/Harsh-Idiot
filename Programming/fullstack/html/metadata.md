
## Metadata

- charset
	- Define character encoding for webpage. .
	- Syntax 
	- ` <meta charset="UTF-8">`
	- prevent encoding errors , supports all languages 
    
- viewport
	- makes webpage responsive 
	- syntax 
	- `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
	- uses for mobile responsive ,proper page scalling 
    
- description
	- short discription for webpage 
	- syntax 
	- `<meta name="description" content="Portfolio Website">`
	- use for SEO ,search engine snippets keep around 150-160 characters
    
- keywords
	- list page keywords 
	- syntax
	- `<meta name="keywords" content="HTML,CSS,JavaScript">`
    
- author
	- use for author 
	- `<meta name = "author" content="Harshith">`
	- for team projects and documentation 
    
- favicon
	- For website icon in browser tab  
	- `<link rel="icon" href="favicon.png>`
    
- Open Graph Tags
	- control link previews on social media 
	`<meta property="og:title">` 
	`<meta property="og:description">`
	`<meta property="og:image">`
	`<meta property="og:url">`

### Best Practices

	- Use one `<h1>` per page.
	- Write meaningful page titles.
	- Add unique meta descriptions.
	- Use semantic HTML.
	- Add `alt` text to images.
	- Keep URLs clean and descriptive.
