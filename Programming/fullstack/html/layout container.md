#  Layout Containers

## `<div>`

**Purpose:** Generic block-level container.

**Syntax**

```html
<div>
    Content
</div>
```

**Use**

- Group elements
    
- Page layout
    
- CSS styling
    
- JavaScript targeting
    

⭐ Most used HTML element.

---

## `<span>`

**Purpose:** Generic inline container.

**Syntax**

```html
<span>Text</span>
```

**Use**

- Style part of a text
    
- Icons
    
- Highlight words
    

⭐ Most used for inline content.

---

## Container Patterns

**Purpose:** Common layout structures used in websites.

**Examples**

- Page Container
    
- Card
    
- Grid
    
- Flex Container
    
- Sidebar
    
- Navbar
    
- Hero Section
    

**Use**

- Organize page layout
    
- Responsive design
    

---

## Wrapper Elements

**Purpose:** Wrap related elements into one container.

**Example**

```html
<div class="wrapper">
    ...
</div>
```

**Use**

- Center content
    
- Apply spacing
    
- Set max-width
    
- Group sections

# ✅ Best Practices

- Use `<div>` for layout and grouping.
    
- Use `<span>` for inline text styling.
    
- Prefer semantic elements (`<section>`, `<article>`, `<header>`) over `<div>` when they describe the content.
    
- Keep containers simple and avoid unnecessary nested `<div>` elements.
